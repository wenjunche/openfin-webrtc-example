# openfin-webrtc-example

A sample WebRTC app that uses signaling and STUN servers hosted by OpenFin.

### Running the sample

1. Install project dependencies.

   `npm i`

1. Build the project.

   `npm run build`

1. Start the Webpack dev server.

   `npm run start`

1. Start the Webpack dev server.

   `npm run start`
